本人胡工于2024年12月12日终于学会git同步个人电脑上代码到线上Github平台仓库，
现在开始使用，线上位于visualmoot/github目录下，个人电脑位于d:/github目录下

个人电脑主要是做一个django项目，管理公司货品
项目名称：vmerp，
目录位置：D:\github\vmerp

1、创建虚拟环境：
在vmerp文件目录窗口地址栏输入cmd,在命令行输入 python -m venv env

2、激活虚拟环境：
在cmd命令行输入D:\github\vmerp\env\Scripts\activate 或  .\env\Scripts\activate

3、在命令行中以(env) D:\github\vmerp>  开头，表明已经进入虚拟环境
创建requirement.txt,将各种依赖包名写入文件中，一行只能写一个包名
后期上线可以使用 pip install -r requirements.txt 来批量安装依赖包
